  1. List four disadvantages of using assembly languages:
    1. Limited portability: Assembly languages are specific to a particular computer architecture, which means that code written in assembly for one processor will not run on another without significant modifications.
    2. Steeper learning curve: Assembly languages are low-level programming languages, requiring programmers to work with machine-specific instructions and register manipulations.
    3. high-level programming languages, which have more intuitive syntax and abstractions. Increased development time: Writing code in assembly language can be time-consuming, as programmers
    4. Difficulty in debugging and maintenance: Assembly code can be challenging to debug and maintain, as it is often difficult to read and understand.
  1. List four expectations programer have for writing programs in a high-level language("third generation" language):
    1. Increased productivity: High-level languages are designed to be closer to natural language, making them easier to read, write, and maintain than low-level languages like assembly.
    2. Platform independence: High-level languages are generally designed to be platform-independent, meaning that code written in one high-level language can be run on multiple platforms without modification.
    3. Abstraction: High-level languages provide abstraction that hides low-level details and complexity from the programmer.
    4. Strong standard libraries: High-level languages typically have strong standard libraries that provide a wide range of pre-written functions and modules that can be used to perform common tasks.

Questions & Answers:

  1. Q.Why does the textbook refer to "the Tower of Babel" regarding computer languages?
    2. A.context of computer languages is an analogy that highlights the diversity and potential confusion that can arise from the numerous programming languages available. The Tower of Babel is a story from the Bible in which people attempt to build a tower that reaches the heavens.
  2. Q.Why isn't there just one computer programming language?
    3. There isn't just one computer programming language because different languages are designed to serve specific purposes, address various problems, and cater to different programming paradigms.
  3. Q.Why didn't FORTRAN use the less than ( < ) symbol when comparing two numbers?
    4. was first developed in the 1950s, computer hardware and character sets were quite different from what we have today. Early computers used various character sets, many of which did not include the less than (<) and greater than (>) symbols that we are familiar with today. One such character set was the IBM 704's punched card code, which was used by the first FORTRAN compiler.
  4. Q.What was COBOL designed for?
    5. was designed primarily for business applications, with a focus on data processing tasks such as data manipulation, handling, and reporting. It was developed in the late 1950s and early 1960s by a joint effort between industry and government, led by Grace Hopper, aiming to create a standardized, portable, and easy-to-read programming language for business use.
  5. Q.What was the language, C, designed for?
    6. The C programming language was designed for system programming, with a focus on writing operating systems, compilers, and other low-level software. Developed by Dennis Ritchie at Bell Labs in the early 1970s, C was created as a successor to the B programming language, which was itself based on the earlier BCPL language.
  6. Q.What is the difference between C and C++?
    7. C and C++ are both general-purpose programming languages, but they have some fundamental differences in terms of their features, programming paradigms, and use cases.
  7. Q.Who or what is the language, Ada, named for and why?
    8. an English mathematician and writer. Ada Lovelace is widely recognized as the world's first computer programmer due to her work on Charles Babbage's early mechanical general-purpose computer, the Analytical Engine.
  8. Q.What is a Java "applet"?
    9. A Java applet is a small, self-contained Java program designed to be embedded and executed within a web browser or an applet viewer. Java applets were introduced in the 1990s as a way to create dynamic, interactive content for websites using the Java programming language
  9. Q.How much does Python cost to download, install, and use?
    10. Python is a free and open-source programming language, which means that it is available for download, installation, and use at no cost.
  10. Q.What is Microsoft. NET?
    11. is a software development platform and framework created by Microsoft for building, deploying, and running applications on Windows and other platforms. The .NET framework provides a common set of tools, languages, and libraries that enable developers to create a wide range of applications, from desktop and web applications to mobile and cloud-based solutions.
  11. Q.What is SQL and what is it used for?
    12. is a standard language used for managing and manipulating relational databases. It is a domain-specific programming language designed for managing structured data in a relational database management system (RDBMS).
  12. Q.What is R and what is it used for?
    13. R is a free and open-source programming language and environment for statistical computing and graphics.
  13. Q.What is Go (or golang) and what is it used for?
    14. Go, also known as Golang, is an open-source programming language created by Google in 2007. Go is a statically-typed language that is designed for building scalable and reliable software applications.
  14. Q.What is Swift and what is it used for?
    15. Swift is a programming language developed by Apple Inc. in 2014 for building software applications for macOS, iOS, watchOS, and tvOS. Swift is designed to be easy to use and learn, while also providing powerful features for modern app development.

Briefly describe each of the following alternative programming paradigms:

  1. Functional programming:
    2. Functional programming is a programming paradigm that emphasizes the use of functions to create software. In functional programming, functions are treated as first-class citizens, meaning that they can be passed as arguments to other functions, returned as values from functions, and stored in variables.
  2. Logic programming:
    3. Logic programming is a programming paradigm that is based on the principles of formal logic. In logic programming, programs are expressed as a set of logical statements, which are used to infer the answers to queries.
  3. Parallel programming:
    4. Parallel programming is a programming paradigm that allows multiple instructions to be executed simultaneously on different processors or processor cores. Parallel programming is used to improve performance and increase the throughput of applications that require a lot of computational power.

  5. Why is "Can someone help me?" not an actual question?
    6. Stack Overflow is a question & answer site. You may think that you have asked a reasonable, answerable question, but a question like "Can someone help me"? really doesn't qualify. That means that a primary characteristic of a good question is that it is likely to be searched by someone else. "Will you help me do X?" doesn't do that. More than likely, no one will ever be trying to do, and searching for, the exact same thing as you in exactly the same way.
  6. Under what circumstances may I add "urgent" or other similar phrases to my question, in order to obtain faster answers?

Nobody is going to see that you need an answer ASAP and then drop everything they're doing in order to help you. Your emergencies are your own. Alternatively: lack of planning on your part does not constitute an emergency on my part. At worst it'll be seen as very rude.
