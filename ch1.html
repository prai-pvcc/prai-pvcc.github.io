Briefly define:

  1. Computer Science

    2. Computer Science is the study of computers and computational systems.

  2. Algorithm

    3. A algorithm is a set of instructions for solving a problem step-by-step to accomplish a task

  3. Sequential Operation

    4. Sequential Operation is a sequence of instructions that caries out a single well defined task then when that task is finished it moves on to the next one.

  4. Conditional Operation

    5. Conditional Operation is "question-asking set of instructions because they ask questions and it moves foward baised on the input.

  5. Iterative Operation

    6. Iterative Operation unlike the other operation repeats a set of instructions until it gets the value that has been set.

  6. Store Program Computer

    7. A computer that stores instructions in its memory to enable it to preform a varity of tasks in a sequence or intermittently.

  7. Programmer

    8. A person who writes, modfies, and tests code that allows computer software and applications to function properly.

  8. Integrated Circuit

    9. A chip with thouands or millions of tiny resistors, capacitros, diodes, and transistors that is able to function as many things like a timer, counter, logic gate, or even computer memory.
