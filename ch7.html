  • Briefly describe or define:
    1. Computer networks
      2. computer network is a set of independent computer systems intercon- nected by telecommunication links for the purpose of sharing information and resources.
    2. Analog device( define and give an example)
      3. Analog device is a mechanical or electronic device that does not use digital signals. While a modern computer works by sharing binary, numerical information, analog devices convey information, like sound or visuals, by storing it on a physical medium like film, tape, or vinyl.
      • Example of analog devices
        1. Vinyl records and record players
        2. Polaroid or film cameras
        3. any clock without a digital display
    3. Digital device(define and give an example)
      4. A digital device is a piece of physical equipment that uses digital data, such as by sending, receiving, storing or processing it.
      • Personal Computers
      • Servers
      • Mobile Devices
      • Entertainment Systems
    4. Modem
      5. A modem (short for modulator-demodulator) is a device that converts digital signals from a computer or other digital device into analog signals that can be transmitted over a telephone line, cable or other communication channels, and vice versa.
    5. Bandwidth
      6. Bandwidth refers to the amount of data that can be transmitted over a network within a given amount of time. It is typically measured in bits per second (bps), kilobits per second (Kbps), megabits per second (Mbps), or gigabits per second (Gbps).
    6. Ethernet
      7. Ethernet is a widely used wired networking technology that allows devices to communicate with each other over a local area network (LAN). It was developed in the 1970s by Xerox Corporation's PARC research center and is now standardized by the Institute of Electrical and Electronics Engineers (IEEE).
    7. WLAN
      8. a wireless local area network (WLAN), a short-distance form of wireless networking, a user transmits from his or her computer, tab-let, or smartphone to a local wireless base station, often referred to as a wireless router, access point, or hot spot, that is no more than a few hun-dred feet away.
    8. IEEE 802.11 Wireless network standard
      9. Wi-Fi, is a set of protocols and standards that define wireless network communications. The standard specifies the methods and technologies that wireless networks use to communicate data between devices, such as computers, smartphones, and IoT devices.
    9. MAN
      10. stands for Metropolitan Area Network, which is a network that covers a geographic area larger than a local area network (LAN) but smaller than a wide area network (WAN). A MAN typically covers a single city or metropolitan area, such as a campus, a business district, or a city center.
    10. Internet of things
      11. refers to the network of physical devices, vehicles, appliances, and other items that are embedded with sensors, software, and network connectivity, enabling them to collect and exchange data over the internet.
    11. LAN
      12. stands for Local Area Network, which is a computer network that connects devices within a limited geographical area, such as a home, office, or school.
    12. Repeater
      13. repeater is a network device that amplifies or regenerates a signal that has weakened over a long distance transmission In telecommunications and networking, signals can become distorted or weakened as they travel over a long distance due to attenuation or interference.
    13. Bridge
      14. bridge is a network device that connects two or more network segments together, allowing them to communicate with each other as if they were a single network.
    14. WAN
      15. stands for Wide Area Network, which is a type of computer network that spans a large geographic area, such as a country or even multiple countries.
    15. Packet and packet switching
      16. packet is a unit of data that is transmitted over a network. It is composed of a header, which contains information about the packet's source and destination, and a payload, which contains the actual data being transmitted.
    16. ISP
      17. stands for Internet Service Provider, which is a company that provides internet access to customers. ISPs typically offer various types of internet services, such as broadband, DSL, cable, or satellite, that allow users to connect to the internet and access online resources.
    17. Firewall
      18. firewall is a network security device that monitors and controls incoming and outgoing network traffic, based on a set of predefined security rules.
    18. Protocol
      19. protocol is a set of rules and guidelines that governs the communication and interaction between devices or systems on a network. In the context of networking, protocols are used to enable the transmission and reception of data between devices, ensuring that the data is transferred correctly and reliably.
    19. Cloud computing
      20. Cloud computing refers to the delivery of computing resources, such as servers, storage, databases, software, and applications, over the internet, on a pay-per-use basis.
    20. Net neutrality
      21. Net neutrality is the principle that all internet traffic should be treated equally by internet service providers (ISPs), without discrimination or preference given to any particular website, application, or content provider.
  • Briefly desctibe the five layers of TCP/IP(The internet protocol hierarchy) :
    • Layer 1. physical layer
      • This layer defines the physical aspects of network communication, including the transmission media, the connectors, and the electrical and optical signals used to transmit data over the network.
    • Layer 2. Data link layer
      • This layer is responsible for defining how data is formatted for transmission over the network and how it is transmitted between two network devices. It also manages the flow of data between the two devices.
    • Layer 3. Network layer
      • This layer is responsible for routing data between different networks. It determines the optimal path for data transmission based on the network topology and other factors.
    • Layer 4. Transport layer
      • This layer is responsible for ensuring the reliable delivery of data between network devices. It handles error detection and correction, flow control, and data segmentation.
    • Layer 5. Application layer
      • This layer contains the protocols that are used by applications to communicate over the network. This includes protocols like HTTP for web browsing, SMTP for email, and FTP for file transfer. It is the layer that most users interact with when they use the internet.
